# インストール

**この手順はエラーなく進む前提で作成されています。**

**もしエラーが発生して先へ進めない場合は、遠慮なくメンターや教材の作成者に報告して助けをもとめてください。**

## インストール

まずこのGitリポジトリをローカルにcloneします。

```shell
cd ~/Dev
git clone ****
```

つぎにcloneしたディレクトリに移動して、次のコマンドを実行してください。

```shell
cd ~/Dev/****
bash install.sh
```

コマンドの実行が完了したら、次へ進んでください。

## 動作確認

次のコマンドで、それぞれのコンテナ(WEBサーバ、APIサーバ、DBサーバ)を起動します。

```shell
docker-compose up
```

### WEBサーバの動作確認

http://localhost:3201/ にアクセスできることを確認してください。

### APIサーバの動作確認

http://localhost:3202/ にアクセスできることを確認してください。

### DBサーバの動作確認

次のコマンドを実行して、下のような結果が表示されることを確認してください。  
(完全に一致しなくても構いません。)

```shell
docker-compose exec db mysql -uroot -proot -e "SELECT * FROM codmon.sample;"
```

以上で動作確認は終了です。
